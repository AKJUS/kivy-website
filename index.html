Welcome to Kivy

Kivy is an open source software library for rapid development of applications that make use of innovative user interfaces, such as multi-touch apps.

Let's whet your appetite! Kivy is:

  • 100% free to use
  • Business friendly: LGPL 3 license
  • Available for Windows, Mac OS X, Linux & Android
  • Able to run the same code on all supported platforms
  • Stable and has a documented API
  • Hardware-accelerated through OpenGL ES 2.0
  • Compatible with most input protocols / devices natively:
    • Tuio
    • WM_Touch, WM_Pen (Most windows devices)
    • Mac OS X multitouch peripherals (Trackpad, Magic Mouse)
    • mtdev, Linux Wacom, Kernel HID Input
  • Easy: API in Python
  • Fast: Many parts written in C using Cython
  • Solid: Regression tests
  • Clean: Styleguide checks
  • Actively developed
  • Community-influenced
  • Easy to extend
  • Easy to use through a graphics API
  • Professionally developed
  • Commercially backed and used

Usage example

See how easy it is to create a simple Hello World application that just shows a clickable button:

from kivy.app import App 
from kivy.uix.button import Button 

class MyApp(App): 
    def build(self): 
        return Button(text='Hello World') 

MyApp().run()
