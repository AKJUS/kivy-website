Home

Kivy is an open source software library for rapid development of applications that make use of innovative user interfaces, such as multi-touch apps.

Let's whet your appetite! Here are some features:

  • 100% free to use
  • Actively developed
  • Stable and documented API
  • Fast: Many parts written in C
  • Easy: API in Python
  • Easy to extend
  • Supports most input devices
  • Easy to use graphics API
  • Hardware-accelerated through OpenGL
  • Same code for Windows, Mac OS X, Linux & Android
  • Professionally developed
  • Community-influenced
  • Commercially backed and used
  • Clean: Styleguide checks
  • Solid: Regression tests
  • Business friendly LGPL license

About Us

Kivy is a community project, but is lead by professional developers with a (mostly) commercial background. Some of them even run companies that use Kivy for their products, so basically they depend on Kivy for their living.

The Core Team

Many people have contributed to Kivy and we're always interested in growing our community. We have a core team that adds new features, manages the project, coordinates contributions and has an eye on the codebase.
