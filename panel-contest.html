Kivy Programming Contest #1

We are exited to announce the 1st ever Kivy programming contest. The Kivy project and community have been growing. You can help us make it even better and score sweet prizes at the same time. Checkout the prizes and rules below to learn about the rules, how to participate, and the prizes you can win.

Goals of the Contest

  • Encourage and give you an opportunity to learn more about Kivy.
  • Build some fun and exciting apps that will run on Android, Linux, Windows, OSX (and iOS).
  • Show to the world the kind of awesome things that can be built with Kivy in a short amount of time while having fun.
  • Improve Documentation based on participants feedback during contest.

Important Dates

December 24th, 2011 Kivy Programming Contest Announced.
January 1st, 2012 Start of Contest, final rules published.

January 31st, 2012, 23:59 GMT Final Submission Deadline.
February 5th, 2012 Winners announced!
